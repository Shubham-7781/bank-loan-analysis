# 🏦 Bank Loan Analysis Dashboard

## 📌 Overview
This project analyzes bank loan data to extract meaningful insights 
about loan applications, approvals, funded amounts, and repayment 
trends using SQL, Excel, and Power BI.

## 🛠️ Tech Stack
- SQL (Data Extraction & Querying)
- Excel (Data Cleaning & Pivot Tables)
- Power BI (Interactive Dashboard)

## 📂 Dataset
Contains 38,000+ loan records including:
- Loan Amount & Funded Amount
- Interest Rates
- Loan Status (Good/Bad)
- Borrower Details
- Regional Data

## 📊 Key Features
- Total Loan Applications tracking
- Monthly Trends (MoM Analysis)
- Good Loan vs Bad Loan Classification
- Regional Analysis by State
- Loan Purpose Breakdown
- Borrower Risk Classification

## 🔍 SQL Queries Used
- Total Loan Applications
- Total Funded Amount
- Average Interest Rate
- Average Debt-to-Income Ratio (DTI)
- Good Loan vs Bad Loan Percentage

## 📈 Power BI Dashboard Includes
- KPI Cards (Total Applications, Funded Amount, Amount Received)
- Line Chart (Monthly Trends)
- Filled Map (Regional Analysis)
- Donut Chart (Loan Purpose)
- Bar Chart (Home Ownership Analysis)

## 📌 Key Insights
- Identified states with highest loan default rates
- Analyzed seasonal trends in loan applications
- Classified borrowers into risk categories
- Tracked month-over-month growth in funded amounts
